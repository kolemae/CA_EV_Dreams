Machine Learning Model

    We split our train and test data on number of sales and removed sales percentage. Originally, we planned to use a multiple linear regression, but chose a Random Forest Regressor for better fit and prediction. The model was run 4 times: merged_demo with and without counties, county_year_merged with and without counties.

    The above images show the top 3 features for each model run. Red indicates real values, and green are predicted values. At a glance we can see the model fits well. We also tried out a stacking regressor, leading us to try Gradient Boosting Regression for better accuracy. Both the Random Forest Regressor and Gradient Boosting Regressor did well at predicting EV sales, with RÂ² values above 0.75 and accuracy scores above 75% (respectively) for all models run.
